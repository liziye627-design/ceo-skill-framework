# CEO Examples

This directory contains example workflows demonstrating CEO orchestration.

## Available Examples

### 1. Simple Web Application
**File**: `simple-web-app.md`

Demonstrates a basic web development workflow:
- PM defines MVP scope
- UI/UX creates design
- Frontend implements
- QA validates

### 2. SaaS Launch
**File**: `saas-launch.md`

Shows a full Build-One lifecycle:
- Phase 0: Feasibility (PM, Growth, Brand)
- Phase 1: Solution (PM, UI/UX, Engineering, Data)
- Phase 2: Delivery (EM, Engineering, QA, Security)
- Phase 3: Launch (Release, Growth, Support, Data)

### 3. Negotiation Example
**File**: `negotiation-example.md`

Illustrates conflict resolution:
- Frontend proposes React Context
- Backend proposes Redux
- Negotiation and arbitration

## Running Examples

To run an example:

```
/CEO I want to build a simple web app

(Paste from examples/simple-web-app.md if needed)
```
