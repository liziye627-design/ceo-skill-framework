# Design Skills

Skills for visual design, user experience, and interface creation.

## ui-ux-pro-max (INCLUDED)

**Purpose**: Comprehensive design system generator

**Status**: Included in this repository

**Capabilities**:
- 67 design styles (glassmorphism, minimalism, brutalism, etc.)
- 96 color palettes
- 57 font pairings
- 25 chart types
- 13 tech stacks (React, Next.js, Vue, Svelte, etc.)

**Usage**:
```
/CEO Create a landing page with glassmorphism design
```

**Installation**:
```bash
cp -r skills/ui-ux-pro-max ~/.claude/skills/
```
