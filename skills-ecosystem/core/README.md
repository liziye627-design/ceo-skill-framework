# Core Skills

Foundational orchestration skills used by CEO for project management and decision-making.

## build-one

**Purpose**: Phase-gated project lifecycle framework

**Description**: Implements a 4-phase delivery pipeline:
- Phase 0: Feasibility
- Phase 1: Solution
- Phase 2: Delivery
- Phase 3: Launch/Ops

**Usage**: Loaded automatically in Build-One track

---

## commercial-pm

**Purpose**: Product management brainstorming playbook

**Description**: Provides structured frameworks for:
- ICP (Ideal Customer Profile) definition
- JTBD (Jobs to Be Done) analysis
- MVP scoping
- Pricing and packaging decisions
- GTM (Go-to-Market) planning
- Metrics and KPI definition

**Usage**: Loaded for Commercial PM track

---

## agent-negotiation-protocol

**Purpose**: Multi-agent conflict resolution

**Description**: Implements structured negotiation:
- Round 1: Initial proposals
- Round 2: Refinement based on feedback
- Escalation to @arbiter if needed

**Usage**: Loaded when roles disagree

---

## self-iteration-loop

**Purpose**: Post-run learning and improvement

**Description**: Analyzes execution results to:
- Identify what worked well
- Find areas for improvement
- Update agent selection scores
- Adjust thresholds

**Usage**: Loaded after execution completes
